#  The Royal Hotel Booking Website

![Preview](assets/preview.png)

A modern and responsive hotel booking website built using HTML, CSS, and JavaScript.

---

##  Features

- Responsive Navigation Bar  
- Hero Header Section  
- Booking Form UI  
- About Section  
- Room Showcase Cards  
- Services Section  
- Explore Section  
- Elegant Footer with Auto-Updating Year  

---

##  Technologies Used

- HTML5  
- CSS3 (Grid & Flexbox)  
- JavaScript  
- Remix Icons  
- ScrollReveal.js  

---

##  How to Run

1. Download or clone the repository  
2. Open `index.html` in your browser
